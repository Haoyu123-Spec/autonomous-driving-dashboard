# RL 深化方案

## 目标
在现有 Dueling DQN 多 AGV 避障基础上，深化强化学习模块，使其更贴近真实自动驾驶决策场景。

## 改进清单

### 1. 动态障碍物场景
- 环境中增加随机移动的障碍物（非智能体，按固定规则运动）
- AGV 需要同时避开其他 AGV 和动态障碍物
- 障碍物数量、速度可配置

### 2. 优先通行机制
- 支持给部分 AGV 设定优先级（如紧急车辆）
- 低优先级 AGV 需要主动避让高优先级 AGV
- 奖励函数中加入优先通行奖惩项

### 3. 网络结构升级
- **NoisyNet**：用噪声层替代 ε-greedy，提升探索效率
- **Prioritized Experience Replay (PER)**：按 TD-error 优先级采样
- 可选：多头注意力捕捉多智能体交互关系

### 4. 训练改善
- 多场景课程学习（简单→复杂渐进训练）
- 更丰富的指标：碰撞率、到达率、平均步数、优先级遵从率
- 训练曲线实时绘制（rewards + 各项指标）
- 模型保存时附带配置信息

## 文件规划
- `dueling_dqn_agv.py` — 原始代码保留
- `rl_advanced/` — 新模块目录
  - `env.py` — 增强版环境（动态障碍 + 优先级）
  - `network.py` — 升级版网络（NoisyNet + PER）
  - `agent.py` — 升级版智能体
  - `train.py` — 训练入口（课程学习 + 多指标）
  - `config.py` — 统一配置

## 执行顺序
1. 先改环境 → 2. 再改网络/智能体 → 3. 改训练流程 → 4. 跑训练验证
